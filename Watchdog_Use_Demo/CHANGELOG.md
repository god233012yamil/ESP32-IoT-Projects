# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.3.0] - 2025-10-02

### Added
- Comprehensive README.md with project documentation
- Mermaid flowchart diagram showing task behavior and system flow
- Detailed explanation of all four demo tasks
- Configuration instructions for stack overflow detection
- Expected output examples and troubleshooting guide
- LICENSE file (MIT License)
- CONTRIBUTING.md with contribution guidelines
- .gitignore for ESP-IDF projects
- sdkconfig.defaults with recommended settings

### Changed
- Updated project documentation to GitHub-ready format
- Enhanced code comments and documentation in main.c

## [0.2.0] - 2025-09-15

### Added
- Tiny-stack task demonstration
- FreeRTOS stack overflow hook implementation
- Stack consumption helper function

### Changed
- Improved TWDT configuration handling
- Enhanced logging for all task states

## [0.1.0] - 2025-09-01

### Added
- Initial project setup
- Healthy task implementation
- Stuck task implementation  
- Flaky task implementation
- Basic TWDT configuration
- Task creation and monitoring
